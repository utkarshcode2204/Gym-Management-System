# Gym Management System

A full-stack Gym Management System built using HTML, CSS, JavaScript, Node.js, Express, and MySQL.

## Features
- Member registration
- Plans and payments
- Membership expiry logic
- Attendance tracking
- Admin dashboard APIs
- JWT authentication

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL

## How to Run
1. Go to backend folder
2. Run `npm install`
3. Run `node server.js`

Backend runs on http://localhost:5000
